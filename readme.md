# How to use

1. download the checkpoint file in https://pan.baidu.com/s/1UxToKiCR4cm2cE2e0o59Fw?pwd=igkk
2. install the requirements
3. modify cuda if not MACOS
4. python stream_tfnet_causal_live.py --list-devices
5. python stream_tfnet_causal_live.py --stereo-device X --output-device Y
   (Replace X and Y for the devices listed above)
